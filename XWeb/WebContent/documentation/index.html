使用

X系统可以通过两种不同的方式使用，通过WWW方式访问。

互联网访问

使用桌面PC访问，屏幕视野宽，便于管理；同时支持大数据量的文件上传。

安装APP

移动终端安装 APP，便于进行数据管理，同时便于订阅、支付。也支持订单对账功能。

支持多浏览器

X系统台支持如下浏览器:

  • IE9+
  • Firefox (latest)
  • Safari (latest)
  • Chrome (latest)
  • Opera (latest)

备注: IE9 不支持变换和动画。
